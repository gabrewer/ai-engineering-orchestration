# ai-engineering-orchestration

This is not a demo repository.

This is where I keep the agents, skills, and orchestration patterns I actually use to build systems with AI.

I’ve spent 30+ years writing software. What’s changing now isn’t just the tools—it’s how we work.

More and more, building software is shifting from writing code to defining intent, constraints, and evaluating outcomes.

This repo is an attempt to make that shift visible and learnable.
